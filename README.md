# **📌 About**

여러 가지 플랫폼의 웹툰에 대한 정보를 제공하며 정보는 여섯시간 간격으로 갱신됩니다.

현재 정보가 제공되는 웹툰 플랫폼은 다음과 같습니다.

- 네이버웹툰
- 카카오웹툰
- 카카오페이지

# **🙏 Request**

웹툰 정보를 요청하는 API에 대한 자세한 설명은 [Swagger 문서](https://korea-webtoon-api.herokuapp.com/api-docs)에서 확인할 수 있습니다.
